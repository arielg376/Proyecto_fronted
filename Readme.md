**Ace of Base Cervecería** es un proyecto de sitio web desarrollado como parte del curso de Desarrollo Frontend con JavaScript.  
El sitio simula una cervecería artesanal con un catálogo de productos, carrito de compras interactivo y un diseño atractivo y responsive.

Incluye las siguientes secciones principales:

-Inicio (`index.html`) → Presentación de la cervecería y catálogo de productos.
-Carrito de Compras → Interfaz dinámica para gestionar pedidos.
-Productos → Galería de cervezas con detalles y precios.
-Contacto → Formulario de consulta (en desarrollo).

El proyecto utiliza HTML5 semántico, CSS3 con Flexbox/Grid, y JavaScript modular para una experiencia de usuario fluida.

# Características Principales del proyecto

-Catálogo de Cervezas:
Visualización completa de todos los estilos de cerveza artesanal,
Diseño en cuadrícula que se adapta automáticamente a cualquier dispositivo,
Cada producto muestra su imagen, nombre, estilo y precio de forma clara.

-Carrito de Compras Interactivo:
Agregar y eliminar productos fácilmente,
Cálculo automático del total de la compra,
Persistencia de datos con localStorage - ¡no se pierde tu pedido al recargar!

-Diseño Responsive:
Maquetación moderna usando CSS Grid y Flexbox,
Compatible con todos los dispositivos: móviles, tablets y desktop,
Tipografía y colores que reflejan la esencia de una cervecería artesanal.

-JavaScript Modular
Código organizado en módulos específicos para cada función,
Manejo eficiente de eventos y actualizaciones en tiempo real,
Arquitectura escalable que permite agregar nuevas features fácilmente.
